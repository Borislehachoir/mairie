# mairie-de-sisco
A la racine de ce projet, il y a deux choses : je voulais trouver un moyen de réaliser un projet qui me permette de développer de nouvelles compétences, et je souhaitais aussi donner un coup de neuf au site de ma mairie, de manière purement bénévole. 

Pour les besoins de ce projet, j'ai utilisé, au 21 novembre 2025, trois languages de programmation front-end (HTML/CSS/JavaScript), plusieurs APIS (dont Sanity Studio, OpenAgenda, Lucide Icons) et plusieurs librairies, notamment Node.js. 

Il y a, au 21 novembre 2025, trois pages globalement prêtes :
- la page "Services Publics";
- la page "Articles";
- la page "Agenda";
